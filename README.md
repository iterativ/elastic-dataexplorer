# Searchkit ES6 Boilerplate
Small project to get started using searchkit. Webpack, es6 (es2015 babel)

## Steps
* npm install
* npm start
* visit http://localhost:3000

## Webpack build

> webpack --config webpack.dev.config.js

# Visit at:

> http://data.iterativ.ch/

# Elastic endpoint:
